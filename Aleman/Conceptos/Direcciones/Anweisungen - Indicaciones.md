## Lokale Präpositionen (preposiciones de lugar)

- in der Nähe -> Para expresar cercanía
- neben -> Para expresar contigüidad
- bei -> proximidad con contexto o conexión

### Beispiele

- =={blue}der== Bahnhof -> **Im** Bahnhof // in der Nähe **vom** Bahnhof // neben **dem** Bahnhof
- =={red}die== Schule -> **In der** Schule // in der Nähe **von der** Bahnhof // neben **der** Bahnhof
- =={green}das== Kino -> **Im** Kino // in der Nähe **vom** Kino // neben **dem** Kino

Im    = in + dem
Vom = von + dem


## Grammatikregeln
- Si la ubicación es una plaza se usa **am** (an + dem)
	- Das restaurant am Friedensplatz 22.
- Si la ubicación es una calle se ocupa **in der o im = in dem**
	- Rod wohnt ==in der== Lindenallee 19. (=={red}die== Allee)
	- Rod wohnt ==im== Wiesen weg 8. (=={blue}der== Weg)
