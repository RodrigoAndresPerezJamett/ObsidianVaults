A la pregunta "Wo" se responde por lo general con una de las nueve preposiciones (estas rigen tanto el dativo como el acusativo)
- ==in / auf / unter /über / vor / hinter / neben / an / zwischen==

Si se ocupa "Wo" se pregunta por un lugar y no por una dirección, bajo este contexto se responde con ==dativo==

| Geschlecht           | Artikel | Dativ       |
| -------------------- | ------- | ----------- |
| =={blue}Maskulinum== | der     | dem / einem |
| =={green}neutral==   | das     | dem / einem |
| =={red}weiblich==    | die     | der / einer |
| Plural               | die     | die / den   |

## Ejemplo

1. Wo sitzt Nico? -> Nico sitzt ==auf einem== Stuhl (=={blue}der== Stuhl)
2. Wo sind die Bilder? -> Die Bilder sind ==zwischen den== Türen (=={red}die== Türen / Plural)
3. Der Stuhl ist ==vor dem== Zimmer (=={green}das== Zimmer)
4. Das Bild ist ==an der== Wand (=={red}die== Wand)


## Tip/s
- A la mayoría de los nombres se añade en plural la terminación -n
- A los nombres ya terminados en -n o -s no hay que cambiarlos
- ==in + dem -> im==
- ==an + dem -> am==
